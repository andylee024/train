# DNT Weightlifting Integration Methodology

How to fold an externally-prescribed Olympic weightlifting program (DNT, or any coach-supplied OLY program) into Andy's arc without breaking the arc's diagnosis-driven structure.

This is the methodology used to integrate DNT's 2-week block (May 17–30, 2026) into Block 1 of the Summer 2026 Dunk arc. Reuse this pattern whenever the coach issues a new block.

---

## Core principle

The arc is built around a diagnosis (strength-dominant, reactivity-deficient → close ESD via power + reactive, not max strength). The coach's OLY program is **strength-speed work** with no reactive content. They serve different goals.

**Don't let the coach's program rewrite the arc.** Treat it as a 2-week specialization mesocycle that swaps in OLY primaries on existing OLY days, leaves reactive/upper/recovery work untouched, and uses the arc's body-part split as the accessory framework.

---

## Five-step integration

### 1. Catalog the coach's program

For each day, separate **lifts** (heavy OLY work, the "practice") from **accessories** (triceps, dips, core, etc.). The lifts are what you're protecting; the accessories can be replaced.

Example from DNT 2-wk block:
- Day 1 lifts: muscle snatch, snatch HP, OHS — accessories: triceps, side bends
- Day 2 lifts: power C&J, front squat — accessories: muscle clean HP, back ext
- Day 3 lifts: hang power snatch, snatch DL — accessories: wrist, strict press
- Day 4 lifts: hang power clean, rack jerk — accessories: bar dip, back ext
- Day 5 lifts: power snatch, snatch DL — accessories: pull-up, good morning
- Day 6 lifts: power C&J, back squat — accessories: core

### 2. Map coach days → arc days

The arc has a fixed weekly shape (Sun lower light / Mon upper push / Tue athletic / Wed recovery / Thu heavy lower / Fri upper pull / Sat jump). Assign each coach day to an arc day where the lift pattern fits the day's theme.

Heuristic:
- Snatch family + OHS → Sun (light overhead, day after Sat jump)
- Heavy snatch + snatch DL → Tue (athletic day, sprint pairing)
- Heavy clean + front squat → Thu (existing heavy lower day)
- Hang clean → Sat (jump-day OLY finisher)

Skip coach days that are redundant (Day 3 vs Day 5 — both heavy snatch + snatch DL) or that fight the arc rationale (Day 6 back squat → max strength is exactly what the arc avoids).

### 3. Apply constraint substitutions

Andy's two non-negotiable constraints:
- **L wrist (De Quervain's)**: no front rack. Substitutions:
  - Front squat → strap-loop grip (loops over bar, fingers hold straps not bar)
  - Rack jerk → push press neutral grip
  - Full power clean → hang power clean (no clean catch slam)
- **R shoulder**: no barbell OHP. Substitutions:
  - Strict press → Landmine press
  - Strict press dropped from primary (substitution exists in Fri arc)

When in doubt, drop the lift before forcing a sub that aggravates either joint.

### 4. Slot accessories into the arc's body-part split

The coach's accessories don't have to land on the coach's prescribed day — they go where they thematically fit Andy's split.

| Coach accessory | Arc day | Why |
|---|---|---|
| Triceps extension, side bends | Mon (upper horizontal) | Push day = press finisher |
| Back extension | Thu (heavy lower) | Pairs with C&J + front squat per coach intent |
| Wrist exercise | Fri (upper vertical) | De Quervain's rehab fits a wrist-mobility day |
| Bar dip, pull-up | Fri (or Wed) | Already in arc — coverage achieved |
| Strict press | Fri Landmine Press | Substitution already in arc |
| Good morning | skipped | Posterior covered by SLDL + RDL + Snatch DL |
| Core (Day 6) | Mon side bends | Already covered |

Drop minor arc finishers (Cable Fly, DB Lateral, Incline Curl, GHR) to make room — they're not load-bearing for any Goal.

### 5. Preserve the untouchables

Even with full coach integration, these stay sacred:
- **Sat jump portion** (sprints + approach + depth + barbell jump) — the goal lives here
- **Mon bench progression** — Goal G2 (bench 260 1RM)
- **Wed deep flex + pull-up volume** — Goal G3 (side split) + G2 pull-up
- **Fri upper vertical arc** — Goal G2 pull-up

If integration would crowd these, integration loses.

---

## Watch list during a coach block

- **Approach touch + CMJ on Sat**: primary KPI. If either drops, intensity is too high — dial coach prescriptions to 75% instead of 80%.
- **Wrist pain (1-10) every DNT day**: especially Sun (OHS), Thu (front squat), Sat (HPC). ≥3 → drop or sub the lift.
- **Top set drop > 5% on bench or squat**: arc cut/load rules still apply.
- **CNS fatigue indicators**: poor sleep, low morning HRV, snatch top set dropping > 5% from session 1 to session 4. Trigger early termination of the coach block.

---

## When NOT to integrate

The methodology assumes the coach's program is at most a 2-week specialization. Reject (or push back to the coach) if:
- The program prescribes >3 OLY days in a 7-day cycle AT 80%+ for >2 weeks (CNS overreach risk on top of arc reactive work).
- The program includes heavy back squat as a weekly fixture (fights arc anti-strength rationale).
- The program prescribes front-rack movements without a workable wrist substitution (Day 2 front squat works with strap-loop; rack jerk doesn't really, push press is OK but isn't the same lift the coach asked for).
- The block runs >3 weeks (move it to its own arc block at that point, not a specialization inside an existing block).

---

## Post-block exit

After the 2-wk DNT block ends, restore the arc's prescribed OLY work for Block 1:
- Tue: Hang Power Snatch back to arc's % prescription
- Thu: Power Clean back to arc's % prescription, Speed Squat returns
- Sat: Hang Power Clean back to arc's lighter % for jump-day pairing

Keep the DNT accessories (triceps, back ext, wrist) integrated if they're working — they're additive without conflict.

Log what worked vs didn't in the end-of-week review. Feed lessons into the next coach block's integration.
