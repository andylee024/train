# Arc Decomposition Model

> The 1-3-1 goal model applied to training program design. Every layer of the program follows the same structure: 1 qualitative purpose → 2-3 quantitative yes/no goals → decompose into children.

## The Pattern

```
Arc (3-6 months)
 └── 1 Purpose: qualitative direction ("peak vertical jump")
 └── 3 Goals: measurable, yes/no testable
      └── Block (3-5 weeks each)
           └── 1 Purpose: what this phase achieves
           └── 3 Goals: measurable, block-specific
                └── Week
                     └── Focus: which quality is emphasized
                     └── Volume/intensity targets
                          └── Day
                               └── Session type + exercise prescription
```

## Rules

1. **Purpose is qualitative.** It describes the direction, not a number. "Convert strength into explosive power" not "increase broad jump by 6 inches."

2. **Goals are quantitative and binary.** Each goal must be answerable with yes or no at the end of the time period. "Squat ≥350 lb" not "get stronger."

3. **Goals at each layer serve the parent's purpose.** Block goals should, when achieved, demonstrably advance the arc's goals.

4. **3 goals maximum.** More than 3 creates competing priorities. If you have 5 goals, you have 0 focus.

5. **Decomposition stops at the day.** A day has exercises with sets/reps/load — it doesn't need its own purpose/goals. The day serves the week's focus.

## How This Drives Programming Decisions

The arc's purpose determines the **training style** (vertical jump, powerlifting, athletic performance). The arc's goals set the **measurable targets** that the testing protocol tracks. The block sequence determines the **periodization model** — which qualities get emphasized when.

At the block level, the purpose determines the **exercise selection emphasis** (strength exercises vs power vs reactive). The block goals set the **progression benchmarks** that decide whether to advance to the next block or repeat.

At the week level, the focus determines the **session split** (which days train which qualities). Volume and intensity targets come from the periodization model.

## Example: Vertical Jump Arc

**Arc Purpose:** Peak vertical jump and dunk a basketball.

- G1: Add ≥4 inches to standing vertical
- G2: Achieve clean 2-hand rim grab from approach  
- G3: Reach ≤186 lb while maintaining squat ≥350 lb

**Block 1 Purpose:** Build power output foundation while maintaining max strength.

- G1: Maintain squat ≥350 lb
- G2: Establish clean technique at ≥155 lb
- G3: Drop to ≤189 lb

**Block 2 Purpose:** Convert max strength into explosive power.

- G1: Hang power clean ≥185 lb for triples
- G2: Standing broad jump ≥9.0 ft
- G3: Drop to ≤186 lb

**Block 3 Purpose:** Peak reactive power and vertical jump.

- G1: Standing vertical +4 inches from baseline
- G2: Successfully dunk or clean 2-hand rim grab
- G3: Bodyweight ≤186 lb

Notice how each block's goals roll up into the arc's goals. Block 1's squat maintenance serves G1 (you need the strength base to convert). Block 2's broad jump target is an intermediate marker toward G1. Block 3's goals ARE the arc's goals — it's where you test.

## Anti-Patterns

- **"Quests" or gamification.** This is not a game. It's a structured goal decomposition. Using quest terminology obscures the planning logic.
- **Goals that can't be tested.** "Feel more explosive" is not a goal. "Broad jump ≥9 ft" is.
- **Blocks without a clear parent relationship.** If you can't explain how Block 2's goals serve the arc's goals, the block doesn't belong.
- **More than 3 goals at any layer.** Prioritize ruthlessly.
