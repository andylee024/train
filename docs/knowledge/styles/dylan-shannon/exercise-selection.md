# Dylan Shannon (POWERJACKED) — Exercise Selection

> **~75 exercises** filtered for the Dylan Shannon style.
> Source: `knowledge/exercises.md` + style-specific annotations from sources/.

This style spans the full force-velocity curve plus high-volume bodybuilding accessories. Unlike pure vertical-jump (which trims aggressively), POWERJACKED is *additive* — it keeps a wide accessory pool because the "look jacked" half of the philosophy demands hypertrophy variety.

## Schema

Each exercise has:

- **training_quality** — `max_strength`, `strength_speed`, `speed_strength`, `reactive`, `skill`
- **muscle_group** — `quads`, `hamstrings`, `glutes`, `calves`, `chest`, `back`, `shoulders`, `arms`, `core`, `full_body`
- **movement_pattern** — `squat`, `hinge`, `lunge`, `push`, `pull`, `olympic`, `plyometric`, `sprint`, `mobility`
- **intensity_tier** — `low`, `medium`, `high`, `max`
- **bilateral** — true/false
- **equipment** — array of required items
- **notes** — *style-specific:* typical loading, when to use, when to skip

## Substitution map

| If unavailable / contraindicated | Substitute | Reason |
|---|---|---|
| Power Clean | Hang power clean → DB clean → KB swing | Preserve hip-extension intent when full clean unavailable |
| Depth Jump (knee pain) | Continuous CMJ from floor | Removes high eccentric stress, preserves SSC |
| Hurdle Jump (no equipment) | Continuous Vertical Jump or Broad Jump | Same reactive demand, no hardware |
| Sprint (no surface) | Bike Sprint, Hill Sprint, Sled Push | Preserves max-effort top-end; loses some specificity |
| Back Squat (back issue) | Front Squat or Pendulum Squat | Quad-dominant variant with less spinal load |
| Hack Squat / Pendulum (no machine) | Heel-elevated Front Squat or DB Bulgarian SS | Preserves quad emphasis |
| Olympic Platform (gym disallows drops) | Hang Power Clean from blocks (no drop required) | Eliminates the catch-and-drop pattern |
| GHR (no machine) | Nordic Hamstring Curl | Same eccentric-emphasis hamstring training |

## Disallowed in this style

| Exercise | Reason for exclusion |
|---|---|
| Long-distance running (zone 2 > 30 min) | Conflicts with the speed pillar; calorie drain crushes hypertrophy |
| Daily heavy squatting (Smolov, Bulgarian-method) | Incompatible with sprint + jump days |
| Pure powerlifting peaking (singles, no accessory) | Strips out the hypertrophy pillar |
| CrossFit-style metcons on lift days | Degrades CNS quality of subsequent compound lifts |
| Single-modality conditioning replacing sprints | "Cardio" doesn't substitute for max-effort sprints — different adaptation |
