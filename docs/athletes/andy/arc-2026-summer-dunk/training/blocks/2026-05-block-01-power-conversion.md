# Block 01 — POWER CONVERSION + UPPER BUILD

## Purpose

Close the explosive strength deficit. Convert existing strength into expressed power. Squat moves to maintenance (1×/wk Sun); jump squats / Olympic lifts / sprints get the volume. Build real upper body base for bench + pull-up. Establish the daily flex protocol.

## Goals

1. **G1**: Standing CMJ +1 inch from baseline AND barbell jump 4×4 @ 33% squat (120 lb) clean (Serves: Arc G1)
2. **G2**: Bench 3RM ≥ 245 lb (from 230) AND weighted pull-up +25 lb × 5 reps clean both sessions (Serves: Arc G2)
3. **G3**: Side split distance reduced ≥ 3 cm from baseline (1 deep day/wk = conservative) (Serves: Arc G3)

## Programming Strategy

1. WEEK SHAPE: Sat = peak performance day (jump). Thu = power buildup. Sun = lightest squat (must NOT compete with Sat 24h later).
2. Squat: TRUE MAINTENANCE — Sun top set 75→80% (275→295 lb), 1×5 + 2×5 back-off lighter. Just keeps the pattern warm; never push.
3. Bench: +2%/wk wave (72→82%, 165→190 lb). Push volume — 4×6 → 4×5 as load climbs. Test 3RM Wk 6.
4. Pull-Up: Fri 4×5 progresses +5 lb/wk (+10 → +25). Tue 4×5 trails Fri by ~5 lb. Wed 3×8 volume at lighter load.
5. Power Clean (Thu): 5×3 wave 70→78% (135→155 lb). Hang Power Clean (Sat) 4×3 at 65→75%.
6. Hang Snatch (Tue): 5×3 wave 55→65% (105→125 lb). DB Push Press neutral 4×4.
7. Speed Squat (Thu): 5×2 wave 55→65% squat 1RM. Jump Squat (Thu) 4×4 @ 25→30%. Barbell Jump (Sat) 4×4 @ 30→33%.
8. Sprints: Sat 3×20yd + 2×30yd full effort (jump day); Tue 3×20yd + 1×40yd finisher (variety day).
9. Reactive: Low-box depth jump (18 in) Sat 2×5 from Wk 1 — early intro since athlete is reactivity-deficient.
10. SLDL/RDL: wave-loaded 4×5,5,4,3, +10 lb/wk (185→225). Same weight all 4 sets within session.
11. Side split: Wed deep work — weighted hold 3×60s, +5 lb every 2 wks (10→20 lb). PNF C-R 3 rounds.
12. CNS spacing: Sat jump → Sun light squat 24h (light only) → Mon upper → Tue speed-strength 48h post squat ✓ → Thu power 48h post Tue ✓ → Sat jump 48h post Thu ✓.
13. Wk 1-2 RPE 6-7. Wk 3-4 RPE 7-8. Wk 5 RPE 8-9 (peak). Wk 6 deload (60% volume, drop intensity 10%).

## Split (6 Workouts / Week, Wed Optional)

1. Sun: Lower Strength (LIGHT maintenance) + Posterior Chain
2. Mon: Upper Horizontal — Bench + Row
3. Tue: Speed-Strength B — Olympic Variety + Sprint Finisher
4. Wed: Recovery + Deep Flex (Optional) + Pull-Up Volume
5. Thu: Speed-Strength A — Olympic Power + Speed Squat
6. Fri: Upper Vertical — Pull-Up + Dips
7. Sat: JUMP LAB — Sprints + Approach + Depth + Loaded Power
