## TL;DR

Run the side split as a proper Matt-Smith **2×/week skill** — a **deep loaded session on Wednesday** (the established leg-recovery day, 72 h before jump day) and a **light active/technique session embedded into Sunday** (6 days before the next jump, kept low-load). Push it hard through **Block 2 (Wk 7–12 = "Build")**, then drop to **maintenance (1×/week) in Block 3 (Wk 13–18)** so loaded adductor work never competes with the dunk peak. This is the best shot at **G3 (−8 cm)** without taxing **G1 (the dunk)**.

The Wed+Sun pairing is, conveniently, one of Matt's own recommended 2×/week spreads, and it keeps every arc constraint intact (no loaded split within 48 h of Saturday).

## Block-by-block plan for the rest of the arc

### Block 2 — Wk 7–12 · **BUILD** (push the split)

- Run **both** sessions every week.
- **Progression = depth first** (go deeper, not heavier). Only add weight to the Wed Side Split hold when the current depth is clean for all sets across **two** sessions. The generated block already ramps the weighted hold 20→30 lb — reframe that ramp as *depth-gated*, not automatic.
- **Re-assess at Wk 9 and Wk 12** using the course's angle measurement (below). If the limiter shifts (e.g., adductor opens but hip-tilt/technique becomes the cap), swap an opener for a technique-biased piece.
- Wk 12 is the block deload/test — drop flex volume ~40 % that week, re-measure.

### Block 3 — Wk 13–18 · **MAINTENANCE** (protect the dunk peak)

- **Drop to 1×/week (Wed only).** Trim Wed flex to opener + compound hold + Kneeling Tilt; Sunday flex becomes optional/light.
- **Why back off when the goal isn't hit yet:** brain (stretch-tolerance) adaptations persist for years; body-side adaptations take ~6 weeks to reverse. A 6-week maintenance window will **hold** the range you built in Block 2 without eroding it — while freeing recovery for the dunk peak (PAP complexes, max dunk attempts). Loaded adductor work is exactly what you *don't* want competing with the peak.
- **Re-test the split at Wk 18** alongside the dunk test.

**Bottom line on G3:** the −8 cm target is realistic *if* Block 2 is the hard push and Block 3 holds. Front-loading the work is what makes it compatible with the dunk.
